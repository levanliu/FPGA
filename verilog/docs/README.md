## 系统特性

1. 支持多协议通信（SPI/I2C/UART）
2. 实时数据采集与分析
3. 可配置测试模式
4. 自动化波形记录功能
5. 跨时钟域同步机制

## 快速开始

```bash
# 查看系统文档
open docs/README.md

# 运行仿真测试
make sim
```
